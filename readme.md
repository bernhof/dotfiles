# Mikkel's dotfiles

Uses [bash-it](https://github.com/bash-it/bash-it) for shell customization.

## First things first

    sudo apt update
    sudo apt install -y git
    git clone git@github.com:bernhof/dotfiles.git ~/.dotfiles
    source ~/.dotfiles/install.sh
